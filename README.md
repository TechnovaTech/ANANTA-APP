# 🚀 ANANTA APP - Complete Project

## 📁 Project Structure

```
ANANTA APP/
├── Ananta_FrontEnd/     # React Native Mobile App (Expo)
└── Ananta_BackEnd/      # Admin Panel (Next.js) + API
```

## 🎯 Components

### 📱 **Mobile App (Ananta_FrontEnd)**
- **Tech:** React Native + Expo
- **Features:** User app with authentication, live streaming, wallet, etc.

### 🖥️ **Admin Panel (Ananta_BackEnd)**
- **Tech:** Next.js + TypeScript + MongoDB
- **Features:** User management, recharge approval, KYC verification
- **URL:** http://localhost:3011
- **Login:** admin@ananta.com / Admin@123

## 🚀 Quick Start

### 1. Setup Admin Panel
```bash
cd Ananta_BackEnd
npm install
npm run seed    # Create admin user
npm run dev     # Start admin panel
```

### 2. Setup Mobile App
```bash
cd Ananta_FrontEnd
npm install
npm start       # Start Expo
```
