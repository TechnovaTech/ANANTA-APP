## 🔄 Timezone Conversion Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                    TIMEZONE CONVERSION LOGIC                     │
└─────────────────────────────────────────────────────────────────┘

Backend (Java):
┌──────────────────────────────────────┐
│ LocalDateTime.now()                  │
│ ↓                                    │
│ JVM Timezone = Asia/Kolkata (IST)   │
│ ↓                                    │
│ Result: 2024-01-15T14:30:00         │
│ ✅ IST time captured                 │
└──────────────────────────────────────┘

Database (PostgreSQL):
┌──────────────────────────────────────┐
│ Hibernate JDBC Timezone = IST        │
│ ↓                                    │
│ INSERT: '2024-01-15 14:30:00'       │
│ ✅ IST time stored                   │
└──────────────────────────────────────┘

Mobile App (React Native):
┌──────────────────────────────────────┐
│ Receive: "2024-01-15T14:30:00"      │
│ ↓                                    │
│ Convert to IST: +5.5 hours offset   │
│ ↓                                    │
│ Format: "15/01/2024  02:30 PM IST"  │
│ ✅ IST time displayed                │
└──────────────────────────────────────┘

Admin Panel (Next.js):
┌──────────────────────────────────────┐
│ Receive: "2024-01-15T14:30:00"      │
│ ↓                                    │
│ toLocaleString({                     │
│   timeZone: 'Asia/Kolkata'          │
│ })                                   │
│ ↓                                    │
│ Format: "Jan 15, 2024, 02:30 PM IST"│
│ ✅ IST time displayed                │
└──────────────────────────────────────┘
```
